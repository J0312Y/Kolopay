# TrustPay Ledger (UI prototype)

Prototype UI React (Vite) for the TrustPay Ledger mobile-like layout.

✅ Stack: Vite + React + Tailwind CSS + lucide-react

## Installation

1. Ouvrez le dossier du projet :

   cd trustpay-ledger

2. Installez les dépendances :

   npm install

3. Lancez le serveur de développement :

   npm run dev

## Notes
- Le composant principal est `src/TrustPayApp.jsx` (votre code fourni).
- Tailwind est déjà configuré. Si vous souhaitez utiliser une police (ex: Inter), installez-la et importez-la.
- Voulez-vous que j'exécute `npm install` et lance le serveur pour vous ?
